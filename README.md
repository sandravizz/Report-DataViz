# IW-Wohnindex Q2 2026 — Interactive Report

An interactive, visualization-first web version of a written report, created from the [Institut der deutschen Wirtschaft (IW)](https://www.iwkoeln.de) analysis. Not a replacement for the PDF, but a different path alongside it — both formats have their reasons to exist. Here, the reader moves through the content while animated charts bring the data to life, with a special focus on the visualizations themselves. Every figure also has a PNG download button, bridging the two formats from a single source.

This edition presents *IW-Wohnindex Q2 2026* (IW-Report 34/2026, 20.07.2026) by Pekka Sagner and Michael Voigtländer — hedonic price trends for German residential property, both purchase and rental. First pass covers Chapter 1 (introduction) and Chapter 2 (the national price index); the regional/city breakdowns and the offer-count special section (Chapters 3-4) are not built yet. Brand derivation and the chart data digitization method are documented in `docs/iw-brand.md`.

This report is built with my interactive-report template; the template itself (tech stack, project structure, how to add chapters and figures) is documented in the README on the `main` branch.
