# IDA 2030: Cliff or No Cliff? — Interactive Report

**Live report: [fdl-report.vercel.app](https://fdl-report.vercel.app/)**

An interactive, visualization-first web version of a written report, created for the [Finance for Development Lab (FDL)](https://findevlab.org). Not a replacement for the PDF, but a different path alongside it — both formats have their reasons to exist. Here, the reader moves through the content while animated charts bring the data to life, with a special focus on the visualizations themselves. Every figure also has a PNG download button, bridging the two formats from a single source.

This edition presents *IDA 2030: Cliff or no cliff?* by Mathilde Barras, Martin Kessler, and Stephen Paduano. The figures step through IDA's balance-sheet growth, the declining equity share financing that growth, the disbursement outlook through 2030 (cliff or no cliff), and IDA's role as the largest source of concessional finance for the world's poorest countries. Client-facing notes — open questions from the kickoff meeting and the FDL brand guidelines — live in `docs/`.

This report is built with my interactive-report template; the template itself (tech stack, project structure, how to add chapters and figures) is documented in the README on the `main` branch.
