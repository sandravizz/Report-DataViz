ScrollySection.svelte pins this whole row (sticky top-0 h-screen) for the height of all its figures stacked. One shared scroll listener computes progress = scroll position through the container ÷ (container height − viewport height), then activeIndex = round(progress × (n − 1)). All three panels read the same activeIndex: the number rail translates + fades neighbours, the chart swaps by opacity crossfade, the description crossfades in step.
